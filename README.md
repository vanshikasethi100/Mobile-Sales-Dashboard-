# Mobile-Sales-Dashboard-
Excited to share a Mobile Sales Dashboard I created using Microsoft Power BI as part of my learning journey. This project helped me dive deep into real-world data analysis and visualization techniques.

Key Learnings & Impact:

1) Built an end-to-end Power BI dashboard for mobile sales analysis, converting raw data into actionable business insights

2) Cleaned, transformed, and modeled data using Power Query, improving data quality and reporting accuracy

3) Created DAX measures and KPIs to track total sales, revenue trends, quantity sold, and top-performing products
   
4) Designed interactive dashboards with slicers and visuals to enable data-driven decision-making for stakeholders

Skills Gained: Data Visualization Data Cleaning. Microsoft Power BI Data Transformation
